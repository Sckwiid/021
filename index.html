📋 Comment utiliser

1
Glisse le bouton ci-dessus dans ta barre de favoris (ou clic droit → Ajouter aux favoris).
2
Ouvre un modèle Meshy, par exemple :
🔗 Grogu sees you v2 — meshy.ai
3
Clique sur le favoris "📦 3D Downloader" depuis la page Meshy. Le panneau de téléchargement s'injecte directement dans la page.
4
Attends quelques secondes que le viewer Meshy charge le modèle, puis clique ⬇ Télécharger dans le panneau.
⚠️ Pourquoi pas une iframe ?
Meshy bloque le chargement en iframe via X-Frame-Options: DENY et une CSP stricte — c'est une protection serveur impossible à contourner depuis un site externe. Le bookmarklet est la seule méthode qui fonctionne sans extension Chrome, car il s'exécute dans le contexte de la page Meshy.

⚙️ Comment ça fonctionne

Le bookmarklet injecte exactement la même logique que l'extension :

🪝
Hook Worker/WASM — intercepte les messages du Web Worker après déchiffrement du modèle GLB.
🌐
Hook fetch — attrape les GLB chargés directement via fetch() depuis un CDN S3/Meshy.
💾
Hook createObjectURL — capture les blobs GLB créés par la page avant qu'ils ne soient consommés.
📥
Dès qu'un GLB est détecté, un panneau flottant apparaît avec le bouton de téléchargement.
